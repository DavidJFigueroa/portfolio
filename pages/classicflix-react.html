ClassicFlix React App

Description

Classic Flix is a movie app created with react. It displays movies that are fetched from an API database, which you can check out here. It lets users register, deregister, update their information, add and delete movies to and from their favorites. It also has a search function and it suggests similiar movies to the user on their profile page. The applications states are handled by Redux.

Languages & Technologies

React, JSX, Bootstrap, Redux

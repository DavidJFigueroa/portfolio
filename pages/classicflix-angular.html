ClassicFlix Angular App

Description

Classic Flix is a movie app created with Angular. It displays movies that are fetched from an API database, which you can check out here. It lets users register, deregister, update their information, add and delete movies to and from their favorites. It shows a database of movies with information skills the movie (desricption, director & genre) to users that are logged in.

Languages & Technologies

Angular, Typescript, Angular Material
